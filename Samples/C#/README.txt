DUO SDK Compilation
------------------------------------
Samples Overview - C# (.NET)
------------------------------------

This project demonstrate simple usage of DUO3D with the C#/.NET Framework.

------------------------------------
Compilation Instructions
------------------------------------

Requirements:

	- Visual Studio 2012
	
	------------------------------------
	Building Samples
	------------------------------------

	1) Install Visual Studio
	2) DUOSamples.sln and build
	3) Run the generated exe file in the bin folder
